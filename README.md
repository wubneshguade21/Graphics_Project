# Graphics Project

# Description
This project is developed using OpenGL and C++.  
It draws the Bahrain flag and the Bluetooth logo with proper graphical transformations.

#Features
- Bahrain flag rendering
- Bluetooth logo drawing
- Transformation applied:
  - Translation (moving objects)
  - Scaling (resizing objects)
  - Rotation (rotating objects)
- Structured and reusable OpenGL code

# Technologies Used
- C++
- OpenGL
- windows.h

# How to Run
1. Open the project in a C++ compiler (CodeBlocks / Visual Studio)
2. Compile the code
3. Run the program
